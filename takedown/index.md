This repository is maintained as a preservation archive. If you are a
rightsholder or authorized representative and want something corrected or
removed, please open a GitHub Issue with the details.

## What to Include

- The URL or file path of the archived item.
- Whether you are requesting a correction, attribution update, or removal.
- A short explanation of the issue.
- A way for the maintainer to verify or follow up if needed.

## Maintainer Response

Requests should be handled respectfully and in good faith. Depending on the
request, the maintainer may:

- Correct metadata or attribution.
- Remove a file from the public archive.
- Replace a local copy with a source link.
- Add clarifying preservation notes.

The goal is to preserve Bill Slawski's work responsibly while respecting the
rights of authors, estates, publishers, photographers, and other rightsholders.
