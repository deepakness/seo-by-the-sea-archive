The [MyBlogLog](http://web.archive.org/web/20070210010548/http://www.mybloglog.com/) widget is one of my favorite widgets for a web site, if not my favorite.

Stephen Pitts, of [SEOPittfall](http://www.seopittfall.com/), has an excellent interview with the CEO of MyBlogLog, Scott Rafer.

If you use MyBlogLog, or might be interested in using it, check out the interview. Steve asks some great questions, and Scott provides a number of details about this site, built around a social networking widget, that I haven’t heard before.

Interested in learning more about where MyBlogLog might be headed? Scott doesn’t reveal everything, but he does tell us some.
