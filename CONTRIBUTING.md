# Contributing

Thank you for helping keep this archive useful and respectful.

Good contributions are narrow, factual, and preservation-focused. The best
changes make the archive easier to browse, verify, or repair without changing
Bill Slawski's voice.

## Good Contributions

- Fix broken internal archive links.
- Fix broken local media references.
- Correct obvious metadata issues in manifests or navigation pages.
- Improve topic indexes and reading paths.
- Improve GitHub Pages accessibility, readability, or mobile behavior.
- Report archive quality issues found during review.

## Please Avoid

- Rewriting Bill's posts.
- Adding commentary inside archived post bodies.
- Adding a broad license to archived writing or media.
- Reintroducing scraping tools or process logs into the public surface.
- Bulk formatting changes that make preservation diffs harder to review.

## Pull Request Checklist

- Keep the change focused.
- Explain what was corrected and how you verified it.
- Do not remove attribution or source URLs.
- Check that GitHub Pages still builds if you changed site files.
- Link to any relevant original source URL, archive path, or issue.
