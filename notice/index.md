This repository is a preservation archive of SEO by the Sea.

Bill Slawski's writing helped many people understand search patents,
information retrieval, PageRank, entities, local search, and the history of
search engines. This archive exists to keep that work discoverable if the
original website becomes unavailable.

## Content Ownership

The archived posts, images, and other media remain the work of Bill Slawski
and/or the appropriate rightsholders. This repository does not claim ownership
of that material.

No new open-source license is granted for the archived writing or media by this
repository. Please do not assume that content in this archive is free to reuse,
republish, remix, or commercialize.

## Source Attribution

The original source is SEO by the Sea:

https://www.seobythesea.com/

Each archived post includes a `source_url` value in its front matter and the
reader-facing site links back to the original source URL when available.

## Purpose

The purpose of this repository is preservation, attribution, and access. It is
not intended to replace the original site, erase authorship, or create a new
licensed edition of Bill's work.

If you are a rightsholder and would like a correction, attribution update, or
removal, please see the [takedown request page]({{ '/takedown/' | relative_url }}).
